# 使用CM API

从2014年Cloudera Manager 4.0开始，引入了API访问功能，用户可以通过API接口，使用编程方式基于Cloudera Manager提供的强大集群管理功能，实现自己的集群管理工具。

API基于Restful风格的HTTP协议，并对其封装成了Java、Python等SDK库，用户可以通过Java程序直接调用。
